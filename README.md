# my_j1 - высокопроизводительная функция Бесселя для Python

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Библиотека `my_j1` предоставляет быструю и точную реализацию функции Бесселя первого порядка j1 для массивов NumPy.
Библиотека использует векторизацию AVX2, инструкции FMA и многопоточность OpenMP, что позволяет обрабатывать массивы в разы быстрее, чем `scipy.special.j1`.

## Особенности

- **Совместимость с NumPy** - функция `my_j1.j1(x)` принимает скаляры и массивы, аналогично `scipy.special.j1`.
- **Точность** - абсолютная ошибка не превышает **2e-15** на всей числовой оси.
- **Производительность** - ускорение ~10 раз по сравнению с SciPy на 2-ядерном процессоре. (см. [бенчмарк](#производительность)).
- **Кроссплатформенность** - работает на процессорах Intel и AMD с поддержкой AVX2. Не требует Intel Compiler или MKL.
- **OpenMP** - эффективная работа на многоядерных системах.
- **Открытый исходный код** - полностью написан на C с Python‑оберткой, без проприетарных библиотек.
